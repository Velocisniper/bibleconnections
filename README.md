This game is like New York Times Connections, but using scripture! 
The game generates a list of verses and chooses 4 non-repeating, uncommon words.

I have added it to github pages at velocisniper.github.io/bibleconnections
Added mobile scaling as well

As someone who barely knows how to code and just supplied the idea, this was written with the help of AI. 
I would love to improve my skill one day to not use AI, but for now it helped me make this the way it is.

The library of bible translations was sourced from https://github.com/bobuk/holybooks/tree/master.
